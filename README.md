# 🤖 RPA Day 5 – Browser Automation using UiPath

## 🚀 Overview

On Day 5 of my RPA journey, I explored **browser automation** using UiPath and implemented real-world use cases. This involved interacting with web applications, extracting live data, and integrating it with Excel for automation workflows.

---

## 🌐 Topics Covered

### 🔧 Browser Automation Activities

Worked with various activities in **UiPath Studio** such as:

* Open Browser / Use Application/Browser
* Navigate To
* Click
* Type Into
* Get Text / Get Full Text
* Send Hotkey
* Delay / Wait for Element

These activities helped simulate real-time user interactions with web applications.

---

## 🌦️ Task 1: Get Current Temperature (Chennai)

* Automated a browser to open Google
* Searched for **current temperature in Chennai**
* Extracted temperature data using **Get Text**
* Stored the result in a variable

---

## 🔁 Task 2: Get Multiple Locations Temperature (Excel + Looping)

* Read multiple locations from an Excel file
* Used **For Each Row** to loop through each location
* Searched temperature for each location in the browser
* Extracted and wrote temperature back into Excel

---

## 🛒 Use Case Project: E-Commerce Data Extraction

### 🎯 Objective

Extract electronic gadget data from two platforms:

* Amazon
* Flipkart

### ⚙️ Workflow

* Open browser and search for a specific electronic product
* Extract product details such as:

  * Name
  * Price
* Store extracted data into Excel:

  * **Sheet 1 → Amazon Data**
  * **Sheet 2 → Flipkart Data**
